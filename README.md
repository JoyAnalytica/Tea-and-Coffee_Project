 Project Summary: Tea & Coffee Shop Database

This project is designed to manage a Tea and Coffee shop using SQL database. The system stores and organizes all necessary information such as products, customers, orders, and payments.

The database includes multiple tables like product details (tea, coffee, snacks), customer information, order records, and billing system. It helps to efficiently track sales, manage inventory, and maintain customer data.

By using SQL queries, the system can perform operations like adding new products, updating prices, recording orders, and generating reports. This project improves data management and reduces manual work in a tea and coffee shop.

Overall, it is a simple and effective database solution for small café or tea stalls.
